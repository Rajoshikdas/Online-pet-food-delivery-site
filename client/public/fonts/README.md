# Fonts

## Rainbow Colors

To use the Rainbow Colors font for "ensures" on the homepage:

1. Download from [FontSpace - Rainbow Colors](https://www.fontspace.com/rainbow-colors-font-f30512)
2. Place `RainbowColors.ttf` (or the .ttf file from the zip) in this folder

If the font file is not present, the site will fall back to Bungee Outline from Google Fonts.
